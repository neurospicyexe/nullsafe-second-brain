# Nullsafe Hearth — Open Thread

**Named by Raziel, 2026-03-09**

A real-time feed/read layer that surfaces live state from the Halseth layer as it relates to the full three-layer stack:

- Halseth: relational record, sessions, deltas, wounds, companions
- Second-brain: searchable knowledge library
- Plural: front state

**Core idea:** Hearth gives a living read of the stack — not a query, not a synthesis, but an ambient awareness of what's happening across all three layers in near real-time.
